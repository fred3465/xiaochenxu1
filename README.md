# 儿童时光相册（微信小程序 Demo）

一个可直接在微信开发者工具预览的三页 Demo：
- 首页（氛围卡片 + 按钮）
- 时间轴（成长记录列表）
- 详情（点赞/收藏交互）

## 小程序预览（微信开发者工具）
1. 用微信开发者工具导入本项目目录。
2. AppID 可用测试号（无账号可先游客模式）。
3. 点击编译即可预览。

> 本项目图片使用文本格式的 SVG 占位图（非二进制），便于在 GitHub/Gitee 直接查看与审阅。

## 网页预览（浏览器）
已提供 `web-preview/` 网页版本，便于直接在浏览器演示交互。

```bash
cd web-preview
python3 -m http.server 5173
```

打开：<http://localhost:5173>
